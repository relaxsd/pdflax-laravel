#Pdflax for Laravel 4

## To install:

```
'providers' => array(
    ...
    
    // Add The Pdflax ServiceProvider
    'Relaxsd\Pdflax\Laravel\PdflaxServiceProvider',
),

'aliases' => array(
    ...
    
    // Add The Pdflax Facade
    'Pdflax'          => 'Relaxsd\Pdflax\Laravel\PdflaxFacade',
),
```
